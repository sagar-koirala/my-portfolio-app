### CubeSat Testing Rig Architecture
Designed and manufactured a custom 2-DoF calibration platform used to test attitude determination and control systems (ADCS) for miniature spacecraft. The platform allows full rotation along both azimuth and elevation axes.

### Electrical Design & Sensor Integration
* **Motor Control:** Utilises the STspin3201 driver with FOC (Field Oriented Control) loops for smooth, high-precision reaction wheel spin speeds.
* **Feedback Loops:** Integrated magnetometer sensors to measure the generated magnetic field strength of magnetic torquers in real-time, alongside Hall-effect encoders for rotor speed feedback.
* **Calibration Suite:** Developed a multi-threaded Python application to visualize data and send test profiles (sinusoidal, steps) to verify controller response curves.
