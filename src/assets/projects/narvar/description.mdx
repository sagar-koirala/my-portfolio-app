### Smart IoT Desk Device Integration
Developed the firmware and system-wide automated workflow for the Narvar Phone Stand, a commercial IoT product consulting engagement. The hardware features an ESP32-S3 MCU which manages device placement sensing and displays a custom UI on screen.

### UI & IoT System Features
* **Custom LVGL Interface:** Designed and optimized a C-based graphical user interface using the Light and Versatile Graphics Library (LVGL), featuring real-time animation state indicators.
* **Geofenced Automation Sinks:** Integrated local limit switches and Time-of-Flight (ToF) sensors to log desk presence. WiFi/MQTT loops publish device states to Adafruit IO, interacting with iOS Shortcuts location triggers to execute custom home/away sequences.
* **External Webhook Integrations:** Engineered webhook triggers passing state variables to IFTTT, executing automated room lighting and calendar status adjustments dynamically.
