### Custom Board Engineering & Layout
A collection of custom multi-layer PCB layouts designed for various industrial and hobbyist projects. The designs cover schematic capture, power budgeting, component footprints creation, routing, and DFM validation.

### Highlighted Board Bring-ups
* **WaiterCall Button PCB:** Designed a ultra-compact circular circuit board utilizing micro-size components and an optimized low-power voltage tree to operate off coin-cell batteries.
* **MTQtester Controller Board:** A multi-layer STM32 controller board integrating motor driver feedback circuits, current monitoring shunts, and magnetic Hall sensors for closed-loop ADCS testing.
* **HMS (Health Monitoring System):** Layout design of a wearable sensor module optimizing noise isolation and signal shielding for sensitive analog health metric capture.
